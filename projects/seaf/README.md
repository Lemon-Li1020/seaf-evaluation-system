# Seaf 项目概况

## 项目定位

Seaf 是一个**智能体工厂**，为用户提供可视化的智能体调用入口。

## 三个平台

| 平台 | 职责 |
|------|------|
| **开发平台** | 智能体基本信息维护；MCP / 知识库 / 插件 / 技能关联管理；多空间隔离 |
| **用户平台** | 访问已发布智能体；对话调用；收藏、查看对话记录、评价 |
| **管理平台** | 智能体/MCP审核；资源管理；数据统计报表；日志管理；用户管理；空间管理 |

## 团队

| 角色 | 人数 |
|------|------|
| 产品经理 | 3 人 |
| 开发人员 | ~20 人 |
| 测试人员 | 5 人 |

**迭代周期**：每月一次

**当前状态**：开发阶段，三个平台分别推进，进度不同步
