# Seaf 竞品分析

## 竞品全景分层

### ToC 智能体助手

覆盖：豆包/文心一言/通义千问/Kimi/智谱清言/讯飞星火/腾讯元宝/DeepSeek（国内）
覆盖：ChatGPT/Claude/Gemini/Copilot/Perplexity（海外）

### ToB 智能体工厂

覆盖：Coze/CoTAM/Dify/FastGPT/LangFlow/n8n/Diffy

### Seaf 定位

**ToB 智能体工厂**——给企业/开发者搭建智能体用，三平台协同（开发侧+用户侧+管理侧）

## 分析维度

| 维度 | 说明 |
|------|------|
| **平台概述** | 定位、背景、目标用户 |
| **核心能力** | 智能体编排、MCP、知识库、插件、工作流 |
| **易用性** | 上手门槛、UI/UX、文档质量 |
| **生态与集成** | 插件市场、API 开放、第三方集成 |
| **定价模式** | 免费/付费/企业版 |
| **用户口碑** | 评价、社区活跃度 |
| **与 Seaf 对比** | Seaf 优势、劣势、机会点 |

## Seaf 平台基准能力

- **三个平台**：开发平台 / 用户平台 / 管理平台
- **智能体编排**：编排 Tab
- **MCP 支持**：发布/下架/审核流程（见 `projects/seaf/docs/mcp-process.md`）
- **Skill 支持**：与 MCP 审核流程一致
- **日志模块**：调用日志 + 日志回放 + Langfuse 执行链
- **知识库**：支持知识库关联
- **插件系统**：支持插件挂载
