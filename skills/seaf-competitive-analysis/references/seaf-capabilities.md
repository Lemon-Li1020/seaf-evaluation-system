# Seaf 平台能力基准

> 本文件记录 Seaf 智能体工厂的已知能力，作为竞品对比的基准。
> 持续更新中，如有出入请指正。

---

## 平台架构

三个平台协同工作：

| 平台 | 职责 |
|------|------|
| **开发平台** | 智能体基本信息维护；MCP / 知识库 / 插件 / 技能关联管理；多空间隔离 |
| **用户平台** | 访问已发布智能体；对话调用；收藏、查看对话记录、评价 |
| **管理平台** | 智能体/MCP/Skill 审核；资源管理；数据统计报表；日志管理；用户管理；空间管理 |

---

## 核心功能

### 智能体管理

- 智能体基本信息维护（名称、描述、图标等）
- 多空间隔离，不同空间资源独立
- 智能体详情页 4 个 Tab：编排、日志、发布、运营

### MCP 支持

- MCP 创建与配置（mcp_config）
- MCP 发布：仅本空间（免审开关）/ 全部空间（需审核）
- MCP 下架：管理侧操作
- MCP 审核流程：待审核 → 通过 / 拒绝

### Skill 支持

- 与 MCP 审核流程完全一致

### 知识库

- 支持上传和管理知识库
- 支持知识库与智能体关联

### 插件系统

- 支持插件挂载到智能体

### 日志与监控

- 调用日志列表（用户名称、会话ID、日志ID、对话问题）
- 日志回放：完整对话记录
- 执行链：Langfuse 链路追踪
- 用户名称来源：`t_user.user_name`

---

## API 能力

### 智能体相关

- `/api/agent/...` — 智能体 CRUD
- `/api/agent/detail` — 智能体详情（含 4 个 Tab 数据）

### MCP 相关

- `/api/mcp/list` — MCP 列表
- `/api/mcp/detail` — MCP 详情
- `/api/mcp/tools/list` — MCP 工具列表

### Skill 相关

- `/api/skill/list` — Skill 列表
- `/api/skill/detail` — Skill 详情
- `/api/skill/tools/list` — Skill 工具列表

## 已知短板（竞品对比视角）

- **插件/MCP 生态**：尚未形成开放的插件市场
- **多渠道分发**：暂不支持飞书/微信/Discord 等第三方渠道接入
- **工作流编排**：详情页「编排」Tab 能力尚未收集，可能较弱
- **运营数据分析**：「运营」Tab 能力尚未收集，BI 能力未知
- **社区与生态**：公开社区、插件商店尚未建立
